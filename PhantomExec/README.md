# Phantom Executor Framework

## Overview

The Phantom Executor is a sophisticated framework designed for educational research into Roblox script injection and anti-tamper circumvention. It provides a structured environment for interacting with the Luau VM and exploring advanced security concepts within the Roblox ecosystem.

## Project Structure

*   `PhantomCore.h` / `PhantomCore.cpp`: Core executor logic, VM interaction, and state management.
*   `dllmain.cpp`: DLL entry point and initialization thread.
*   `Phantom_Executor_Design.md`: Detailed architectural design and technical specifications.

## Building and Usage

This project is designed for C++ compilation (e.g., using Visual Studio). The resulting DLL should be injected into the Roblox client process using a compatible injector.
